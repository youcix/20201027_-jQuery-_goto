# 屬性說明

【 須先在前往的元素上新增id 】

屬性名稱        |   屬性說明
---------------|----------------
data-gt-target | 欲前往的目標元素
---------------|----------------
data-gt-duration |持續時間，單位為毫秒
---------------|----------------
data-gt-offset | 位移，往上位移的值
